- [x] Minimize the Difference B/w Heights : ```O(N)```

- [x] Merge Intervals
    - Without using stack
    - TC : O(n logn)
    - SC : O(1)
    - ```Arrays.sort(intervals, (a, b) -> Integer.compare(a[0], b[0]));```
