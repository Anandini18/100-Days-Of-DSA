# Day 17
- [x] Find median of a matrix
  - Without using the extra space
    - TC : ```O(32*R*log(C))```
    - SC : ```O(1)```
  - By using the extra space 
    - TC : ```O(M*N log(M*N))```
    - SC : ```O(M*N)```


- [x] Find the row with maximum no. of 1's
    - Brute Force
        - TC : ```O(m*n)```
        - SC : ```O(1)```
    - Binary Search
        - TC : ```O(m log(n))```
        - SC : ```O(1)```
    - Brute Force
      - TC : ```O(m+n)```
      - SC : ```O(1)```
